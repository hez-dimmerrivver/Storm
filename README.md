# Storm Vinyl
Jan 2026, Jiang He Zhang(Hez)
## Description
An interactive digital vinyl that creates wind sounds from storm videos.
<img width="1000" height="564" alt="Storm Vinyl" src="https://github.com/user-attachments/assets/7f71915c-b7d2-4a3e-9cd0-921efe340951" />


## Concept
The spinning motion of storms reminded me of vinyl records, which inspired this project. It transforms radar videos from storms in 2025 into record-like visualizations. Pixel colors in the radar videos are analyzed to generate different sound pitches and frequencies. The sidebar displays the storms, allowing users to select one, play it like a vinyl record, and hear wind-like sounds.

## Technology Used
This project was mainly developed using P5.js, with the data structured in JSON..

## How to Play
1. Open the website in your browser.
2. Select a storm from the left sidebar.
3. Press the play button to start the wind sound.
4. Pause the sound or select another storm to listen.

## How to Run Locally
1. Clone the repository
2. Install dependencies:
   ```
   npm install
   node server.js
3. Open http://localhost in your browser

## Requirements
- Node.js 
- npm
- Browser with Web Audio API support (Chrome, Firefox recommended)
- Internet connection (if not running locally)
